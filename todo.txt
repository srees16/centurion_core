User says: "Find oversold NIFTY 50 stocks with positive news sentiment and suggest a trade"
Step 1: planner agent → decomposes into 4 sub-tasks

Step 2: Data Ingestion Agent
        → Fetches NIFTY 50 real-time quotes
        → Returns 50 UnifiedQuotes

Step 3: Strategy Scanner Agent (depends on Step 2)
        → Runs RSI scan → finds 6 oversold stocks
        → Returns 6 Signals

Step 4: News Sentiment Agent (depends on Step 3, parallel-capable)
        → Scrapes news for those 6 stocks
        → Returns sentiment: 3 positive, 2 neutral, 1 negative

Step 5: Orchestrator → filters to 3 stocks (oversold + positive sentiment)

Step 6: Risk Guardian Agent
        → Checks position limits → approves 2, blocks 1 (sector overweight)

Step 7: Orchestrator → presents 2 trade recommendations to user
        → User approves → Execution Agent places orders

7. Full audit trail in agent_runs table using langfuse or free dashboard
